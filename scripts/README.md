# Scripts Directory

このディレクトリには、AWS Client VPNインフラストラクチャのセットアップと管理に必要なスクリプトが含まれています。

## スクリプト一覧

### generate-certs.ps1
Windows PowerShell用の証明書生成スクリプトです。easy-rsaを使用してCA証明書、サーバー証明書、クライアント証明書を生成します。

### generate-certs.sh
Git Bash/WSL用の証明書生成スクリプトです。easy-rsaを使用してCA証明書、サーバー証明書、クライアント証明書を生成します。

## 使用方法

詳細な使用方法については、各スクリプトのヘッダーコメントを参照してください。

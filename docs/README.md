# ドキュメントディレクトリ

このディレクトリには、AWS Client VPNインフラストラクチャに関する各種ドキュメントを格納します。

- **IAM Identity Center設定手順書** (`iam-identity-center-setup.md`)

- **証明書配布手順書** (`certificate-distribution.md`)

- **VPN接続手順書**
  - PC用VPN接続手順 (`vpn-connection-pc.md`)

- **トラブルシューティングガイド** (`troubleshooting.md`)
  - よくあるエラーと解決方法

  - 証明書更新手順
  - セキュリティスキャン手順

## 注意事項

- すべてのドキュメントは日本語で記述してください
- 機密情報（パスワード、秘密鍵、APIキー）は記載しないでください
- スクリーンショットを含める場合は、機密情報をマスキングしてください
